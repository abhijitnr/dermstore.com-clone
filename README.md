## Functionality implemented

- Navbar
- Home page
- Home page slider
- Product slider
- Search function
- Register or signup
- Login
- Add to cart
- Remove from cart
- Footer
- All the functionality like add to cart, remove from cart, seraching product all are done using backend. And Backend is build using MongoDB, Redux, NodeJS and ExpressJS.
